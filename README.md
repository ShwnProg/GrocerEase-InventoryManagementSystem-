Grocery Inventory Management System

This project is a simple Inventory Management System for a grocery store, developed as part of our Information Management course. The system is designed to efficiently manage products, categories, and stock levels, ensuring accurate tracking of inventory.

Purpose

The main goal of this system is to:

Track and manage grocery products
Monitor stock levels (stock in/out)
Organize items by category
Provide a simple and user-friendly interface for inventory operations
🛠️ Features
Product Management (Add, Edit, Delete)
Category Management
Stock Monitoring
Basic CRUD Operations
Simple and clean user interface
